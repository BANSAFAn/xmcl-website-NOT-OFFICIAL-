## ⚠️ Disclaimer

This is an **unofficial** XMCL website. The original launcher [xmcl.app](https://xmcl.app) was created by developer **CIO10**. This project is an enhanced version of the website, created to expand functionality and improve user experience.

## 🌐 About the Project

This project is a website for X Minecraft Launcher (XMCL), optimized for both PC and mobile devices. The site provides information about the launcher, usage guides, a blog with news and updates, and the ability to download the launcher for various operating systems.

## 💻 Technologies

This website is built with a modern technology stack:

- **Frontend Framework**: React 18.2.0 with functional components and hooks
- **Language**: TypeScript 5.0.2 for type safety and better developer experience
- **Build Tool**: Vite 4.4.5 for lightning-fast development and optimized production builds
- **Package Manager**: Bun 1.0.2 for ultra-fast dependency installation
- **Styling**: TailwindCSS 3.3.3 for utility-first CSS and rapid UI development
- **UI Components**: 
  - Radix UI for accessible, unstyled components
  - Framer Motion 10.16.4 for beautiful animations and transitions
- **Routing**: React Router 6 for client-side navigation
- **State Management**: TanStack Query (React Query) for data fetching and caching
- **Content Rendering**: React Markdown with plugins for rich content display
- **Icons**: Lucide Icons for consistent, beautiful SVG icons
- **Accessibility**: ARIA-compliant components with keyboard navigation support

## 🚀 Getting Started

### Prerequisites

- Node.js 16+ or Bun 1.0+
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/BANSAFAn/xmcl-website-NOT-OFFICIAL-.git

# Navigate to the project directory
cd xmcl-website-NOT-OFFICIAL-

# Install dependencies with npm
npm install
# OR with Bun (recommended for faster installation)
bun install

# Start the development server
npm run dev
# OR with Bun
bun run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the website in your browser.

### Building for Production

```bash
# Build the project
npm run build
# OR with Bun
bun run build

# Preview the production build locally
npm run preview
# OR with Bun
bun run preview
```

### Project Structure

- `/src` - Source code
  - `/components` - Reusable UI components
  - `/pages` - Page components
  - `/utils` - Utility functions
  - `/hooks` - Custom React hooks
  - `/api` - API endpoints
- `/public` - Static assets
